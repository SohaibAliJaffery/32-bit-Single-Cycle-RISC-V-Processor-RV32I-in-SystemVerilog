# 32-bit Single-Cycle RISC-V Processor (RV32I) in SystemVerilog

Version: v1.0

This project is a single-cycle RV32I processor written in SystemVerilog. It includes separate modules for the ALU, control unit, register file, immediate generation, instruction memory, and data memory, along with a testbench and memory initialization files for simulation and verification.

## Files

- `alu.sv`
- `controller.sv`
- `data_memory.sv`
- `immediate_generator.sv`
- `instruction_memory.sv`
- `RamSp.sv`
- `reg_file.sv`
- `rv32i_defs.svh`
- `rv32i_singlecycle.sv`
- `testbench.sv`
- `instruction_memory.mem`
- `data_memory.mem`

## Version

This repository snapshot is tagged as `v1.0`.
